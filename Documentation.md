## NxtUp : Problem Recommendation Model

Initial Idea : A CF API Tracker, that gives user statistics and performance -> Along with a tracker for famous Codeforces sheets like 
               Mostafa Saad Ibrahim's Famous Junior Sheet, A2OJ Sheet

### Revised Long term Idea 

  #### Retained Features 
  1. Codeforces API
  2. User Statistics/ Heatmap for Submission

  #### New Elephant in the room :
    Question recommendation model, that identifies gaps in problem solving by analyzing past 50 submission. 
    No. of WA and TLE, topic, difficulty, rating are the parameters
    Designs a 5 Question Problem Set to bridge the "gap"
    -> It would use an API of LLM to study the gap and identify topic, not just (+200/-200) rated suggestions
